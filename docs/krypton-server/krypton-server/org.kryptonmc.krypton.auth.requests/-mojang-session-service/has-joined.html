Notify the session server that the player with the specified username has joined the server.

This is the only request required to authenticate a user with Mojang.

The serverId field is calculated by hashing the server's ID in ASCII, the shared secret and the encoded version of the server's public key, then digesting that to hex by creating a BigInteger from the MessageDigest.digest function, then converting it to a base 16 string (see MessageDigest.hexDigest)

Return

a Call representing the result of making this request

Parameters

username

the username of the user we want to authenticate

serverId

the server ID, see above

ip

the server's IP
