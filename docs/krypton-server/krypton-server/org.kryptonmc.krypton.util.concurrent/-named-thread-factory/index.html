A thread factory that automatically formats the specified nameFormat by replacing any "%d" keys with the current thread number.

For example, if you have 5 threads in a pool, and the name format is "My Thread #%d", the first thread will be named "My Thread #1", the second will be named "My Thread #2", and so on.
