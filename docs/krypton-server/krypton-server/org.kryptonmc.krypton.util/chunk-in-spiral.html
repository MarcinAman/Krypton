fun chunkInSpiral(id: Int, xOffset: Int = 0, zOffset: Int = 0): ChunkPosition
Content copied to clipboard

Calculates a chunk position from a given id in a spiral pattern.

This algorithm was previously part of Krypton (https://github.com/KryptonMC/Krypton), however it was removed after it was no longer used.

Algorithm:

Given n, an index in the squared spiral p, the sum of a point in the inner square and a, the position on the current square

n = p + a

Credit for this algorithm goes to davidonet (for the original algorithm), and Esophose (for the Kotlin conversion and modifications)

See here for original

Return

a ChunkPosition containing the calculated position in the spiral.

Parameters

id

the id in the spiral

xOffset

an optional X offset

zOffset

an optional Z offset
